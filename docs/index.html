Hbs

A small assistant script repository. This GitHub Pages site provides a public URL and quick usage instructions.

Quick start

Clone the repo, install dependencies, and run the safe assistant:

python3 -m pip install -r requirements.txt
python3 hbs_safe.py --noninteractive

Full interactive assistant

Run the interactive assistant (not recommended in Codespaces):

python3 hbs.py

Notes

  • `hbs_safe.py` will NOT auto-install system binaries or run installers.
  • To use the full capability you must install ollama and run its server: ollama serve &

Custom domain (apex)

This site is configured to use the apex custom domain HBSsitemaker.com. To make the domain live, add the following DNS records at your registrar for the apex domain:

  • A records (point @ to): 
    185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
  • Optionally add a CNAME for www pointing to jayscrypto999-hash.github.io if you want www.HBSsitemaker.com as well.

After DNS updates, GitHub Pages will provision HTTPS automatically (may take a few minutes to hours). Keep `docs/CNAME` in the repo so GitHub associates the custom domain with the Pages site.
